Odoo 10 Point of Sale Extension

Allow users to reserve order by paying initial amount.


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Reserve Product By Paying Initial Amount.
  • Do a Partial Payment for normal order.
  • View Order History.
  • Cancel Reserved Order.
  • Cancel Particular Product From Reserved Order.
  • Pay Order Amount Partially.
  • Manage Customer's Credit Limit.
  • Add Delivery Date In Order.

How to use

  • Use Order reservation functionality by enabling configuration.


  • Manage Customer credit limit and see order history of customer.




  • Turn on Reservation mode to make this order as reserve order.


  • Reserve Order by paying initial amount.


  • Do partial payment without Reserve order.


  • Set Delivery Date.


  • View Reserved Orders.


  • Cancel order by marking products.


  • View Payment history of particular order.
